Dashboard de Análisis de Ventas – Power BI

Proyecto personal enfocado en el análisis de ventas mediante Power BI.

¿Qué muestra este dashboard?
- Ventas totales, transacciones y ticket promedio
- Análisis de ventas por mes (tabla calendario)
- Ventas por categoría y género
- Filtros interactivos por año

Herramientas utilizadas
- Power BI
- DAX
- Modelado de datos
- Excel

Principales insights
- Las categorías Electronics y Clothing concentran la mayor parte de las ventas
- Se observan picos de ventas a mitad de año y en el último trimestre
